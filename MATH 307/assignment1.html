Problem 4 (2 marks)

Find the smallest (positive) integer $N$ such that the Vandermonde matrix for data points $t_0=0,\dots,t_N=N$ has condition number greater than $10^{10}$. Save the result as N4. Use np.vander to create the Vandermonde matrix. See the documentation. Use the function np.linalg.cond to compute the condition number. See the documentation.

In[25]:
# YOUR CODE HERE
threshold = 1e10

for N4 in range(1, 50):  
    t = np.arange(N4+1)               
    V = np.vander(t, increasing=True) 
    cond = np.linalg.cond(V)
    if cond > threshold:
        N4 = N4
        break
